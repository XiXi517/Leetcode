# 21 合并两个有序链表

## 1 题目

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

**示例：**

```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```



## 2 迭代法

### 2.1思路

定义：定义一个新的链表n，和一个哨兵guard指向新链表的最后一个节点；

过程：比较L1，L2的头结点值，将较小的节点链接到新的链表中；

​	    节点被选中后指针L1或L2向后移动；

​	     哨兵向后移动；

终止条件：L1或L2存在一个为null

输出：将非空的链表插入到新链表的后面，返回新链表

### 2.2 代码

~~~java
public ListNode mergeTwoLists(ListNode l1, ListNode l2){
        ListNode n = new ListNode(-1);
        ListNode guard = n;
        if(l1 == null)
            return l2;
        if(l2 == null)
            return l1;

        while (l1 != null && l2 != null){
            if(l1.val <= l2.val){
                guard.next = l1;
                l1 = l1.next;
            }
            else {
                guard.next = l2;
                l2 = l2.next;
            }
            guard = guard.next;
        }
        if(l1 == null){
            guard.next = l2;
        }
        else {
            guard.next = l1;
        }
        return n.next;
    }
~~~

关键点：

​	1.循环终止条件  while (l1 != null && l2 != null)

​	2.特殊条件：当链表为空时做法

​	3.输出：n.next
